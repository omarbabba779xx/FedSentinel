🔐 Gradient Commitment Scheme (Sigma Protocol)
NOT formal ZKP
SHA-256 + HMAC commit-challenge-respond scheme. Provides binding + hiding. Does not use zk-SNARK/STARK circuits. For formal ZKP: integrate circom + snarkjs + Groth16.
1. Commit
c = SHA256(Δw ‖ nonce)
Client binds gradient. Server cannot alter.
2. Challenge
ch = random_bytes(32)
Server issues random challenge.
3. Respond
σ = HMAC(sk, ch‖c)
Server verifies HMAC + recomputes hash.
